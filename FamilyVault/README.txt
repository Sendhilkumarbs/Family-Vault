Family Vault - Portable Edition
================================

One secure place to store and access your family's bank accounts,
documents, property records, and more.

REQUIREMENTS
  Windows PC only. No Java or Maven install needed — Java is included.

QUICK START
  1. Double-click StartApp.bat
  2. Browser opens at http://localhost:8080
  3. Use StopApp.bat to stop the app

FILES
  family-vault-1.0.0.jar  - Application
  runtime\                - Bundled Java (do not delete)
  data\                   - Your saved records (created on first run)
  config\                 - Settings (created when you use Settings page)
  logs\                   - Application logs

MOVE TO ANOTHER PC
  Copy this entire FamilyVault folder to the other computer.
  To bring your existing data, copy the data\ and config\ folders too.

HIDDEN START (no window)
  Double-click StartAppHidden.vbs instead of StartApp.bat

VIEW LOGS
  Double-click ViewLogs.bat
